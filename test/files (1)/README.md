# C# ソースコード学習・生成ツール

C#のソースファイルを読み込んでAIに学習させ、学習したコーディングスタイルやパターンに基づいて新しいコードを生成できるコンソールアプリケーションです。

## 機能

1. **ソースコード学習**: 単一ファイルまたはフォルダ全体のC#ソースコードを読み込み、Claude AIで分析
2. **学習内容表示**: 学習したコードベースの特徴や分析結果を表示
3. **コード生成**: 指示に基づいて、学習したスタイルに沿った新しいコードを生成
4. **ファイル保存**: 生成したコードをファイルに保存可能

## 必要要件

- .NET 8.0以降
- インターネット接続（Claude API使用のため）

## ビルド方法

```bash
cd CSharpLearner
dotnet build
```

## 実行方法

```bash
dotnet run
```

## 使用方法

### 1. ソースコードの学習

メニューから「1」を選択し、学習させたいC#ファイルまたはフォルダのパスを入力します。

```
ファイルパスまたはフォルダパスを入力してください: /path/to/your/project
```

フォルダを指定した場合、そのフォルダ内のすべての`.cs`ファイルが再帰的に読み込まれます。

### 2. 学習内容の確認

メニューから「2」を選択すると、AIが分析したコードベースの特徴が表示されます。

### 3. コード生成

メニューから「3」を選択し、生成したいコードの指示を入力します。

例:
```
指示: ユーザー管理クラスを作成して、CRUD操作のメソッドを含めてください
```

AIは学習したコーディングスタイルやパターンに従って、指示に沿ったコードを生成します。

生成されたコードは画面に表示され、ファイルとして保存することもできます。

### 4. 学習内容のクリア

メニューから「4」を選択すると、学習した内容をクリアして新しく学習し直すことができます。

指示: UserRepositoryと同じインターフェース設計でProductRepositoryを作成してください

## 注意事項

- このツールはClaude APIを使用します（APIキーの設定は不要ですが、claude.aiのアカウントが必要です）
- 大量のファイルを読み込む場合、処理に時間がかかることがあります
- 生成されたコードは必ず確認・テストしてから使用してください
- 機密性の高いコードを学習させる場合は注意してください
